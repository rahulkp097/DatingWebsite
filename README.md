## Key Technologies:

- **Socket.io:** Powering real-time interactions for a dynamic user experience.
- **ZegoCloud:** Enhancing video call quality for meaningful connections.
- **Mapbox:** Geolocation features amplifying user profiles.
- **Razorpay:** Facilitating secure transactions for premium features.
- **Front-end Frameworks:** Tailwind CSS, Daisy UI, Chakra UI - Merging aesthetics with functionality.

## Key Features:

- **Sign In with Google:** Effortless entry into the dating scene.
- **Signup with OTP:** Secure onboarding with OTP verification.
- **Password Hashing:** Prioritizing user security with hashed passwords.
- **Email Notifications:** Timely updates for matches, messages, and important information.
- **Filter Profiles, Search Profiles:** Advanced filters and search for an ideal match.
- **Profile Recommendations:** Intelligent matchmaking based on user profiles.
- **Send/Accept Requests, Shortlist Profiles:** Streamlined connection management.
- **Subscription Plans:** Seamless unlocking of premium features.
- **User Report Option:** Empowering users for a safe community.
- **Chat with Notification, Video Calls:** Real-time communication with added convenience.

## Admin Side:

- **Graphs:** Visualizing user count and revenue trends.
- **User Listing:** Efficient management of user profiles.
- **User Block/Unblock:** Positive community environment maintenance.
- **User Activities:** Monitoring user interactions for enhanced security.
- **Manage Subscriptions:** Streamlining subscription management.
